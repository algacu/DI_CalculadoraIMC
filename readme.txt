Desarrollo de Interfaces - Calculadora IMC

Actividad evaluable para la asignatura de Desarrollo de Interfaces, en la que tuve que programar y desarrollar el interfaz de una calculadora de índice de masa corporal, mediante el uso de estados y componentes en React Native.

Lenguajes de programación:
Javascript (entorno React Native).

Herramientas:
Visual Studio Code.
Android Studio.
Simulator (Xcode).
